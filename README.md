# SQL Assignments Archive

A curated archive of SQL coursework demonstrating core relational database concepts and hands-on query development. This repository includes assignments covering schema design, table creation, constraints, data manipulation, joins, common table expressions (CTEs), window functions, triggers, and stored procedures/functions.

## Overview

This repository is intended to showcase practical SQL skills developed through academic assignments. The work emphasizes both query correctness and database design fundamentals, including normalization-oriented table structure, referential integrity, and procedural database logic.

## Topics Covered

- DDL operations (`CREATE TABLE`, `ALTER TABLE`, `DROP TABLE`)
- Primary keys and foreign keys
- Data insertion and updates
- Multi-table joins
- Common table expressions (CTEs)
- Window functions such as `DENSE_RANK()`
- Constraint management
- Trigger creation
- SQL and PL/pgSQL functions
- Simple business-rule automation inside the database
- And more!

## Example Assignment Content

The uploaded sample assignment includes work such as:

- creating and restructuring book/category tables
- enforcing referential integrity with primary and foreign keys
- writing a genre-based ranking query using CTEs and `DENSE_RANK()`
- building account/check withdrawal tables
- creating trigger-based balance and overdraft logic
- computing dynamic membership status from purchase behavior
